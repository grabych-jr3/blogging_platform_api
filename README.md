# Blogging Platform API
[Blogging Platform API](https://roadmap.sh/projects/blogging-platform-api) - is a simple RESTful API with CRUD operations.

## :hammer_and_pick: Technologies
+ Java 25
+ Spring Boot
+ Redis
+ PostgreSQL

## :rotating_light: Endpoints
+ /posts - create new blog post; return all blog posts
+ /posts?term=tech - return all blog posts that have the term "tech" in their title, content or category
+ /posts/{id} - return a blog post by id
